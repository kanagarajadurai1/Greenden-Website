# GreenBloom Plant Show 2026 🌿

**“Where Nature Meets Innovation”** — An interactive, modern website showcasing the GreenBloom Plant Show 2026. Built with **HTML, TailwindCSS, AOS, Typed.js, Lightbox, and Particles.js**, the site combines aesthetic appeal with immersive interactivity for an engaging user experience. ✨

## Key Features

- ✔️ **Hero Section with Typewriter Effect** – for event highlights 🌟  
- ✔️ **Parallax Backgrounds & Glassmorphism Cards** – modern, immersive UI 🌿  
- ✔️ **Floating Falling Leaves Animation** – adds a natural touch 🍃  
- ✔️ **3D Buttons & Gradient Text** – interactive and visually appealing 🎨  
- ✔️ **Scroll Progress Bar** – enhances navigation feedback 📊  
- ✔️ **Responsive Navbar & Mobile Menu** – includes dark mode 🌙  
- ✔️ **Lightbox Gallery** – showcases featured plants elegantly 🖼️  
- ✔️ **Event, Contact & Feedback Sections** – styled with glass cards 🏷️  
- ✔️ **Particles.js Background** – creates a lively, interactive atmosphere ✨  
- ✔️ **Wave Divider SVGs** – smooth transitions between sections 🌊  

## Technology Stack

- **Frontend:** HTML, CSS, TailwindCSS 💻  
- **Animations & Effects:** AOS, Typed.js, Particles.js ⚡  
- **Media:** Lightbox.js for image galleries 🖼️  
- **Responsive Design:** Mobile-first approach with dark mode toggle 📱
